Get [] [Username v] ::reporter extension

Info
This lets you get info about a user
Added in
Version 1.0
Options
Username
This gets the username
ID
This gets the ID
Avatar URL
This gets the Avatar URL
Gravatar ID
This gets the Gravatar ID
Profile URL
This is the link to the users GitHub profile it looks like
https://github.com/(username)
User Type
This is the type of user
Most accounts are user or organization
Admin
This is if they are a GitHub admin
Name
The user's name
Company
The user's company
Website
The user's website
Location
The user's location setting
Email
The user's email (null if not public)
Bio
The user's bio
Public Repo Count
Count of the user's public repos
Public Gist Count
Count of the user's public gists
Followers
Count of how many users are follwing this user
Following
Count of how many users this user is following
Created At
When the user was created
Updated at
When the user was updated

Needs to have API key set: NO
Example
when gf clicked ask [Who do you want to know about?] and wait say (join [Their name is: ] (Get (answer) [Name v] ::extension)) for (1) secs say (join [Their website is: ] (Get (answer) [Website v] ::extension)) for (1) secs say (join [They work for: ] (Get (answer) [Company v] ::extension)) for (1) secs
